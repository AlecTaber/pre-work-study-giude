HTML

  • The head element contains information about the webpage
  • The body element represents the visual content shown to the user
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport
  • The title element embeds client-side scripts in the HTML document
  • The link element connects an external resource to the HTML document
  • The base element defines the base URL for a webpage
  • The style element defines the CSS styles associated with a website
  • The script element embeds client-side scripts in the HTML document

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A declaration contains two components, the CSS property we want to apply and the value of the property
  • The rel attribute specifies the relationship between the current document and the linked document resource
  • The href attribute specifies the location (URL) of the external resource

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A : adds modifications in the current working branch to the staging area
  • git commit -m: a message that describes what the commit contains and updates it
  • git push: pushes the changes we have locally to our remote GitHub branch
  • git pull: receives a branch's modifications into the local environment

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is a way of introducing decision-making into our code
  • An array is a single variable that is used to hold a group of data
  • An index is a location number to find a variable inside an array
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
  • A function is a set of instructions that tells the computer how to perform a certain task
